# Spotify Karaoke

**Spotify Karaoke** is a browser extension that enhances your Spotify Web Player experience by adding **Romanized** and **Translated** lyrics options.

## Features

*   **Original Lyrics**: View lyrics as provided by Spotify.
*   **Romanized Lyrics**: Automatically transliterate lyrics into Roman characters (great for K-Pop, J-Pop, etc.).
*   **Translated Lyrics**: Translate lyrics into 100+ languages using Google Translate.
*   **Lyrics Caching**: Cache Translated Lyrics to improve performance and reduce API calls.
*   **Sleek Design**: A dark-themed component that blends perfectly with Spotify's native UI.

## Usage

1.  Open [Spotify Web Player](https://open.spotify.com).
2.  Play a song and open the **Lyrics** view.
3.  You will see new control buttons at the top of the lyrics pane: **Original**, **Romanized**, **Translated**.
4.  Click **Romanized** to see the pronunciation.
5.  Click **Translated** to see the meaning in your language.
6.  Click the extension icon in your toolbar to change the **Target Language** for translations.

## Privacy Policy
This extension does not collect any user data. All translations are performed via client-side calls to Google Translate. Settings are stored locally in your browser.
